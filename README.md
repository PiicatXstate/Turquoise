## 小说一句
本项目是一个本地运行的EPUB阅读工具。当然，这并非重点，主要是为了解决阅读文言文的困难，后续将开发丰富的辅助功能，例如可进行查词和AI翻译，更可以写注解或者译文，提高文言文的阅读效率（也是我原先项目ClassicTrans的升级版）  
以上皆为展望，项目刚刚起步  

## 使用技术栈
Vue3 ( 框架 ) + Vite ( 构建工具 ) + Element-Plus ( UI组件库 )
